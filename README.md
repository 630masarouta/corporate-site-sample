# Corporate Site Template - Web制作サンプル

## 🌐 デモサイト
https://630masarouta.github.io/corporate-site-sample/

---

## 📌 概要
地域企業を想定したコーポレートサイトの
静的Webサイト制作サンプルです。

実務案件を想定し、
「企業の信頼感・情報整理・拡張性」を重視して制作しています。

---

## 🎯 現在対応可能範囲
- 静的コーポレートサイト制作（HTML / CSS）
- レスポンシブ対応（スマートフォン最適化）
- テキスト・会社情報の更新
- 事業内容セクション追加
- 実績紹介追加
- 軽微なデザイン修正
- 既存HTML/CSSの修正対応

---

## 💻 使用技術
- HTML5
- CSS3
- モバイルファースト設計
- レスポンシブ設計（768pxブレークポイント）
- Gitによるバージョン管理
- GitHub Pagesを用いた静的サイト公開

---

## 🏗 ページ構成
- Hero（メインビジュアル）
- About（私たちについて）
- Services（事業内容）
- Works（実績）
- Message（代表挨拶）
- Company（会社概要）
- Contact（お問い合わせ情報）

---

## ✨ 実装ポイント
- セマンティックHTML構造
- 拡張しやすいセクション設計
- カードレイアウトによる情報整理
- アクセシビリティ配慮（alt属性・focus対応）
- 保守性を意識したクラス設計

---

## 🔧 今後拡張可能な機能
- お問い合わせフォーム追加
- CMS化（WordPress対応）
- SEO基礎対応
- 多言語対応
- サーバー本番環境へのデプロイ対応
